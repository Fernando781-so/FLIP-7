Este proyecto tiene como propósito recrear el popular juego de mesa "Flip 7" mediante una aplicación de software con arquitectura Cliente-Servidor, desarrollada completamente en el lenguaje Java.

El repositorio está diseñado para facilitar el trabajo en equipo (colaboración a distancia) y para ser compartido con otros usuarios interesados en jugar o en la estructura del código.

Como usarlo:
-Navega el directorio.
-compila y ejecuta los archivos Servidor.java y Cliente.java (en ese orden).
-El Cliente intentará conectarse al servidor.
-Lo siguente sera leer y seguir pasos de lo que quieres hacer.
